Sécurité

Sauvegarde éditoriale

Le backup JSON couvre les contenus éditoriaux. Le backup ZIP couvre tous les fichiers du dépôt visibles dans la branche chargée.

La restauration ZIP remplace l’état du dépôt par le contenu de l’archive. À utiliser après avoir vérifié que l’archive correspond bien au site cible.

Publication

Génération et envoi sur GitHub

La publication écrit les fichiers content/, régénère index.html, pages/*.html, data/articles.json, data/categories.json et data/search-index.json, puis envoie les fichiers modifiés sur GitHub.
